# Diagramas exportados a PNG

Acá van los `.png` exportados de los `.drawio` de la carpeta padre (para presentaciones,
informes y el cliente).

## Cómo exportar (manual, ~30 seg por diagrama)
1. Abrir el `.drawio` en [app.diagrams.net](https://app.diagrams.net) (Open from Device)
   o pedirle al agente que lo abra.
2. **File → Export as → PNG…**
3. Opciones recomendadas: `Zoom 100%` · `Border 10` · ✅ *Include a copy of my diagram* (off) · fondo blanco.
4. Guardar acá con **el mismo nombre** que el `.drawio`:

| Fuente (.drawio) | Exportar como |
|---|---|
| `../calidad-lata-bpmn.drawio` | `calidad-lata-bpmn.png` |
| `../calidad-lata-arquitectura.drawio` | `calidad-lata-arquitectura.png` |
| `../calidad-lata-wf1-ingesta.drawio` | `calidad-lata-wf1-ingesta.png` |
| `../calidad-lata-wf2-procesamiento.drawio` | `calidad-lata-wf2-procesamiento.png` |
| `../calidad-lata-wf5-reporte.drawio` | `calidad-lata-wf5-reporte.png` |

> Si un `.drawio` cambia, re-exportar su PNG para que no queden desfasados.
